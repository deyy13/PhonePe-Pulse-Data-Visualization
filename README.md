# PhonePe Pulse Data Visualization and Exploration

## Overview
This project analyzes and visualizes PhonePe Pulse data using Power BI to uncover insights related to digital transactions, user growth, transaction trends, and regional performance across India.

The dashboard provides an interactive experience with filters, KPIs, maps, and trend analysis to help understand the growth of digital payments in India.

## Domain
FinTech

## Tools & Technologies
- Power BI
- Data Visualization
- DAX
- Excel / CSV Dataset
- Geo Mapping
- Data Analytics

## Problem Statement
The PhonePe Pulse GitHub repository contains large volumes of transaction and user data. The objective is to transform and visualize this data in an interactive dashboard that enables users to explore trends, growth patterns, and business insights.

## Features
- Interactive India Geo Map
- State-wise Transaction Analysis
- User Growth Analysis
- Transaction Type Breakdown
- Year-on-Year Growth Analysis
- Quarter-wise Trend Analysis
- CAGR Calculation
- KPI Cards
- Brand-wise User Analysis
- Dynamic Filters and Slicers

## Key Insights
- Telangana records the highest transaction amount.
- Maharashtra has the highest number of registered users.
- Peer-to-Peer payments contribute the largest share of transactions.
- Significant transaction growth is observed between 2021 and 2022.
- Digital payment adoption has increased rapidly across Indian states.
